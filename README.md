# simple-docker-playground
docker with node server + nginx proxy server

筆記：
https://hackmd.io/s/SJZq1jr8-

## 啟動

1. 下載源碼

```
$ git clone https://github.com/tri613/simple-docker-playground.git myproject
```

2. 到專案根目錄下

```
$ cd myproject/
```

3. 執行 docker-compose

```
$ docker-compose up -d
```

4. Done!
